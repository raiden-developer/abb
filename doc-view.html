Подзаголовок

Для фильтрации списка данных по коду ABB используйте, пожалуйста, параметр manufacturerCode (код производителя). Для получения данных по коду производителя используйте параметр searchField, указав значение manufacturerCode . Для получения данных в облегченном варианте укажите значение simple-json для параметра format. Для выгрузки списка всех данных в формате CSV файла укажите значение csv для параметра format

Классы

Каждый класс может иметь набор допустимых свойств. Например, класс «Устройство защитного отключения (УЗО)» описывается набором из более 20 свойств, среди которых, например: «Количество полюсов», «Номинальное напряжение», «Номинальный ток утечки», «Тип тока утечки» и т.д.
